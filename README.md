# Zomato Restaurant Analysis (Bengaluru)

## Project Overview
This project analyzes restaurant data from the Zomato dataset for Bengaluru to understand customer preferences, pricing trends, and restaurant performance.

The project includes:
- Data Cleaning using Python (Pandas)
- Exploratory Data Analysis
- Power BI Dashboard with 6 report pages
- Business insights and recommendations

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Excel
- Data Visualization

## Dashboard Pages
1. Overview Dashboard
2. Location Analysis
3. Ratings & Services Analysis
4. Restaurant Types & Cost Analysis
5. Best Restaurants & Top Chains
6. Insights & Business Recommendations

## Key Insights
- Online delivery increases average ratings.
- Premium restaurants have the highest ratings but are limited in number.
- BTM, Koramangala, and Indiranagar are top restaurant clusters.
