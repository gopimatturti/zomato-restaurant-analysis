# 📊 Zomato Restaurant Analysis – Bengaluru

An end-to-end Data Analysis project using **Python and Power BI** to analyze restaurant trends in Bengaluru based on the Zomato dataset.

This project explores customer engagement, restaurant performance, pricing trends, and service impact using data-driven insights.

---

# 🧾 Project Overview

Bengaluru has one of India's most competitive restaurant ecosystems with over **40,000 restaurants**.  
This project analyzes restaurant data to understand:

- Restaurant density across locations
- Customer engagement through ratings and votes
- Pricing trends and cost categories
- Impact of services like online delivery and table booking
- Top performing restaurant chains
- Business insights and recommendations

# 🧹 Data Cleaning (Python)

The dataset was cleaned using Python before visualization.

Key cleaning steps:

- Removed duplicate records
- Standardized rating values
- Converted cost and votes into numeric format
- Handled missing values
- Created derived fields:
  - `rating_category`
  - `cost_category`

Final cleaned dataset was used for Power BI visualization.

# 📈 Key Insights

- **65% of restaurants offer online delivery**
- **Table booking restaurants have the highest ratings**
- **Premium restaurants are only 0.2% of the market**
- **BTM, Koramangala, and Indiranagar are the most competitive restaurant hubs**

---

# 💡 Business Recommendations

For Restaurant Owners:

- Enable online delivery & table booking
- Focus on high-demand locations
- Maintain pricing between ₹400 – ₹1500

For Investors:

- Premium dining is under-supplied
- Microbreweries and café lounges show strong potential
